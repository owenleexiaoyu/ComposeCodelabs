# ComposeCalculator

这个项目是参加 [Android Jetpack Compose Study Jam](https://www.gdgcn.net/?page_id=477) 进阶挑战赛的作品。

使用 Jetpack Compose 实现一个简单的计算器。实现的功能如下：

- 支持四则运算
- 适配横竖屏
- 适配深色模式
